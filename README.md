## 格式支持
- 所有歌词与歌词翻译均采用`UTF-8`编码的`LRC`格式。
- 时间戳精度为毫秒。
- 时间戳和歌词之间有一个空格。
### 格式示例
```lrc
[ti: 歌曲标题]
[ar: 歌手名称]
[al: 专辑名称]
[by: 翻译作者（如有）]
[00:01.386] 第一句
[00:08.523] 第二句
[00:16.337] 第三句
```
